# Микросервис обработки и отдачи изображений отелей

Проект разработан в рамках производственной практики студента 1 курса направления «Информатика и вычислительная техника» (ИВТ).

## 🚀 Функционал
* **Ресайз и конвертация**: Изменение размера изображений «на лету» и перевод в формат `WebP` с целевым уменьшением веса ≥ 30% [1.2].
* **Файловое кэширование**: Повторные (горячие) запросы отдаются мгновенно без повторного задействования CPU [1.2].
* **Инвалидация кэша**: Поддерживается автоматический сброс по времени (TTL 24 часа) и принудительная очистка через `DELETE`-запрос [2.4, 2.5].
* **Валидация данных**: Двухуровневая проверка входящих файлов (по расширению и по внутренней структуре структуры JPEG/PNG) [1.21].
* **Контейнеризация**: Полная готовность к деплою через Docker и Docker Compose.

## 🛠️ Стек технологий
* **Язык**: Python 3.11
* **Фреймворк**: FastAPI (Uvicorn)
* **Графика**: Pillow

## 💻 Локальный запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `python -m uvicorn main:app --reload`

## 🐳 Запуск через Docker Compose
Развертывание всей инфраструктуры одной командой:
```bash
docker compose up -d --build
```

## 📊 Примеры API-запросов
* **Получить картинку (GET)**: `http://localhost:8000/image?w=300&h=200&fmt=webp&src=test.jpg`
* **Очистить кэш (DELETE)**: `http://localhost:8000/image?src=test.jpg`
* **Интерактивная документация (Swagger)**: `http://localhost:8000/docs`
